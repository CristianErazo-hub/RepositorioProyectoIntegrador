Se implementó el diagrama de clases bajo arquitectura **MVC**:

*   **Model**: Representa la entidad `Insumo`.
*   **DAO**: Gestiona la persistencia en **MySQL** mediante **JDBC**.
*   **Controller**: Maneja la lógica de negocio.
*   **Vista**: Se implementa en consola.

El proyecto fue estructurado con **Maven** para gestionar dependencias como el conector de MySQL.
